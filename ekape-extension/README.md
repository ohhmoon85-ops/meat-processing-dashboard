# EKAPE 원패스 자동화 Chrome 확장프로그램

로그인 후 **열람용 확인서 버튼을 자동으로 클릭**하고, 선택에 따라 자동 인쇄까지 처리합니다.

## 설치 방법

1. Chrome 주소창에 `chrome://extensions` 입력
2. 우측 상단 **"개발자 모드"** 토글 활성화
3. **"압축해제된 확장 프로그램 로드"** 클릭
4. 이 `ekape-extension` 폴더를 선택

## 사용 방법

1. 대시보드에서 개체 선택 → **"원패스 열람용 확인서 열기"** 클릭
2. 열린 EKAPE 탭에서 **로그인** (한 번만 하면 세션이 유지됩니다)
3. 확장프로그램이 자동으로 **열람용 확인서 버튼을 클릭**
4. 확인서 페이지에서 인쇄

## 설정 옵션

Chrome 우측 상단 퍼즐 아이콘 → "EKAPE 원패스 자동화" 클릭

| 옵션 | 기본값 | 설명 |
|------|--------|------|
| 열람용 확인서 자동 클릭 | ON | 검색 페이지에서 확인서 버튼 자동 클릭 |
| 확인서 자동 인쇄 | OFF | 확인서 팝업 열리면 자동으로 인쇄 시작 |
| 인쇄 후 탭 닫기 | OFF | 인쇄 완료 후 탭 자동 닫기 |
| 클릭 전 대기 시간 | 2.0초 | 페이지 로드 완료 후 클릭까지 대기 |

## 주의 사항

- **자동 인쇄** 옵션은 기본적으로 꺼져 있습니다. 인쇄 전 확인이 필요한 경우 켜지 마세요.
- 확장프로그램이 버튼을 찾지 못한 경우 우상단에 경고 메시지가 표시됩니다.
- EKAPE 홈페이지 UI가 변경된 경우 동작하지 않을 수 있으며, 이 경우 수동으로 클릭하세요.
